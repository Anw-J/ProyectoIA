# Algoritmo A* para calcular la ruta

Proyecto desarrollado en la asignatura de Inteligencia Artificial, que dado una version reducida del metro de CDMX calcula la mejor ruta aplicando el algoritmo A*

## Instalación
Para poder ejecutar el programa, deberán disponer de las dependencias Flask y networkx, para ello ejecutar en terminal:
```bash
python -m pip install flask networkx
```

## Ejecución
Deberán situarse en la carpeta src del programa, que es donde está el punto de entrada main.py, para ejecutar:
```bash
py main.py
```
